## Quick start

### 1. Install a component

```bash
npx shadcn@latest add https://design.nyuchi.com/api/v1/ui/button
```

### 2. Set up theming

Copy the Five African Minerals CSS custom properties into your `globals.css`. See the [Theming guide](/docs/theming) for the full token set.

### 3. Use the component

```tsx
import { Button } from "@/components/ui/button"

export function MyPage() {
  return <Button variant="default">Get Started</Button>
}
```
